Verification Help

This is help for verifying in # ✅・verify and using Double Counter.
Select your issue below to get detailed instructions.

VPN Detected

Your connection may appear to be using a VPN depending on certain network factors. Double Counter blocks networks connected to VPNs.

Fixes:

  • Disable ad-blockers or browser extensions that modify network requests.
  • On iPhone/iPad: Disable "Hide IP Address" in Safari and turn off iCloud Private Relay.
  • Switch from privacy browsers (Opera/Brave) to Chrome or Edge.
  • Check for shared/proxy IPs from your ISP using ipinfo.io.

Detected as an Alt

Your account shares an IP with another Discord account. If possible, please use that account. Otherwise, if your account is terminated, forgotten password or just can't, open a support ticket.

Crawlers Not Allowed

This block may occur if:

  • Using Discord's in-app mobile browser — use your default phone browser instead.
  • Using Cloudflare WARP, DNS modifiers, or similar — disable them and retry.
  • Browser extensions or proxy settings alter your fingerprint — temporarily disable them.

Other Issues

  • Discord is banned in your country.
  • None of the previous steps work.
